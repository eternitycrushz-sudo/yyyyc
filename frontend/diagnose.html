Troubleshooting Guide

No products found?

  • Ensure MySQL is running: net start MySQL80
  • Import initial data: Open MySQL and run dy_analysis_system.sql
  • Or run crawler to fetch data: Use crawler.html page

Getting 401 Unauthorized?

  • Login first using default credentials: admin / admin123
  • Token will be saved in localStorage automatically

Backend connection failed?

  • Ensure Flask backend is running on port 5001
  • Run: python backend/app.py

Steps to get data:

  1. Start MySQL service
  2. Import database: mysql -u root -p dy_analysis_system < dy_analysis_system.sql
  3. Start backend: python backend/app.py
  4. Login with admin / admin123
  5. Navigate to products to see data
